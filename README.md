# Lu System Official

Repositorio do site oficial da Lu System.

## Estrutura

- `index.html`: pagina inicial do site.
- `styles.css`: estilos da pagina.
- `Johnny.PNG`: imagem inicial usada na apresentacao.

## Publicacao

Este projeto esta pronto para ser publicado como site estatico pelo GitHub Pages.
No GitHub, ative Pages usando a branch `main` e a pasta raiz do repositorio.
